\## Project Description

This project is part of Homework 2 in the Development Tools course.

The goal of the project is to practice working with Gradle and writing unit tests in Java using JUnit.



The unit tests were written for the methods in the `App` class.



\## Technologies Used

\- Java

\- Gradle

\- JUnit 5

\- Git

\- GitHub



\## How to Clone the Project



To clone the repository, run:



```bash

git clone <repository-url>

```



Then enter the project folder:



```bash

cd 26b-10142-unittesting-gali\_shir\_oneg

```



\## How to Run the Tests



Make sure Java 21 is installed.



To run all unit tests on Windows, use:



```bash

gradlew.bat test

```



On Linux or Mac, use:



```bash

./gradlew test

```



\## Test Files



The unit tests are located in:



```text

app/src/test/java/org/example/AppTest.java

```



The tested code is located in:



```text

app/src/main/java/org/example/App.java

```
